# NEInduction

## Summary

入职前练习培训项目，主要使用[NEJ框架](https://github.com/NEYouFan/nej-framework)进行开发。

### 技术栈

1. NEJ
2. NEI
3. Koa2
4. Pug
5. MongoDB

## Run

首先需要安装Mongodb，Mongodb连接配置文件为[config](/config/default.js)，运行脚本在[package.json](/package.json)文件中。

```
# 启动mongoDB
npm run mongo
# 启动Node.js服务器
npm run dev
```
